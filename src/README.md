# **`src/`**

This directory contains the packages that make up the Enhanced SAM 3D Body Library.

Each package includes its own `README` providing:

1. A high-level overview of the corresponding project component.
2. The differences between the current implementation and the original project.
3. The optimisations introduced to improve inference speed and reduce overhead.
4. The internal directory structure of the package.

> **Notes:**
> 1. All implementation files contain detailed inline comments explaining backend-specific decisions and non-obvious logic.
> 2. For a deeper exploration of the system architecture and conceptual documentation, refer to [docs/](../docs/README.md).
